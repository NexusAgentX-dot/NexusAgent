# Frontend Brief For Claude

You own the frontend and UI for this hackathon project.

## Product
NexusAgent Lite is a multi-agent commerce and execution network on X Layer.

It should feel:
- sharp
- protocol-native
- premium
- not generic SaaS

## User Story To Visualize
An external agent is brought into the system.
The user submits a concrete intent.
Specialized agents collaborate.
The system records a payment event, executes, and settles on X Layer.

## Screens To Prioritize
1. Landing page
2. Onboarding surface
3. Main workflow dashboard
4. Settlement proof panel

## Visual Priorities
- make the 4 agents feel distinct
- make the flow easy to scan
- make the X Layer settlement proof feel real and important
- show protocol stack: MCP, A2A, x402, Onchain OS, X Layer
- avoid purple-on-white generic crypto AI aesthetics
